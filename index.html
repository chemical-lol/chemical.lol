Detection

Will anti-cheat engines flag it?

We operate entirely in kernel space with randomized memory layouts. Signatures are regenerated on boot and we track emerging heuristics daily. If a vendor flips a new rule, you’ll see an alert in the client before they do.

Hardware

What rigs are supported?

Windows 10/11, DirectX 11 GPUs, and 4GB RAM is the baseline. The loader auto-detects virtualization, TPM status, and secure boot toggles so you never have to reflash anything.

Setup

How long does onboarding take?

Download, run as admin, confirm the hardware fingerprint. The injector stages itself in under 10 seconds and you can swap profiles without relaunching.

Privacy

What data leaves my machine?

None. There is no telemetry layer, no analytics SDK, and no call-home packets. Updates are signed and distributed via diff bundles you can inspect locally.
